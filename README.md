
# **Bot-DmMessage**

**Bot-DmMessage** is a simple Discord bot written in Python that sends direct messages (DMs) to all members of a server in an automated and controlled way.

It is useful for:

* Important announcements
* Event notifications
* Server-wide messages
* Administrative communication

The bot loops through all server members and sends each one a private message, saving time and avoiding manual spam.
